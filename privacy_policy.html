3. Data collected/processed by collector/processor

3.1. Websites and infrastructure

For obvious reasons everything intended by user or bot to be stored on the platform like comment under post can be stored on the server and if that was the intention — presented on the site indefinitely. In case users wants some content they uploaded to be removed they shall contact the owner listed above.

3.1.1. Diagnostics data

The underlying infrastructure stores anonymized diagnostics data about connections and requests made. IP addresses collected have last octet of masked (i.e. 192.168.0.171 becomes 192.168.0.0). This data is needed for stability and security of the platform, which is necessary.

3.1.2. Data location

While the main server is located in the European Union, some services used to deliver and back up websites are hosted in the United States. All of them however are GDPR compliant.

3.2. Traffic analytics - plausible.io

Domains listed above use plausible.io traffic analytics platform. Plausible Analytics collects only aggregated information, which does not allow us to identify any visitor to our website.
Full details of what's collected is available here: https://plausible.io/privacy.

3.3. Content Delivery Network - Cloudflare

Domains listed above use Cloudflare Content Delivery Network. This means all traffic goes also via their servers and some content including scripts and cookies may be injected. This service is required for platform stability and security, which is necessary.
Privacy policy can be accessed at https://www.cloudflare.com/privacypolicy/ - sections that apply to end user.

3.3.1. Cookies used

  • __cfduid - necessary; to identify trusted web traffic

4. What to do if you don't want your data being processed

You can disable cookies in your browser (example tutorial: https://www.wikihow.com/Disable-Cookies) and disable JavaScript using web browser extensions like NoScript (https://noscript.net/).
